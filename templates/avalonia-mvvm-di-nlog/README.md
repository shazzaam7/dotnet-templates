# MyCustomTemplate

A cross-platform desktop application built with Avalonia UI.

## Getting Started

```bash
dotnet restore
dotnet build
dotnet run
```

## Project Structure

```
source/
├── MyCustomTemplate/        # Main application
├── MyCustomTemplate.Core/    # Core business logic
tests/
└── MyCustomTemplate.Tests/   # Unit tests
```
